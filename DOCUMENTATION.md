### Функція celsuis_to_fahrenheit
``````python
def celsius_to_fahrenheit(celsius):
        return (celsius * 9/5) + 32
``````

**Параметри:**
- 'cesius' (float): температура

**Повертає:**
- 'float' - Цельсії переведені в фаренгейти

**Приклад:**
``````python
print(celsus_to_fahrenheit(10)) # Виведе: 50.00°F

### Функція fahrenheit_to_celsus
``````python
def fahrenheit_to_celsius(fahrenheit):
        return (fahrenheit - 32) * 5/9
``````

**Параметри:**
- 'fahrenheit' (float): температура

**Повертає:**
- 'float' - Фаренгейти переведені до цельсіїв

**Приклад:**
``````python
print(celsus_to_fahrenheit(10)) # Виведе: 50.00°F
``````

### Функція celsius_to_kelvin
``````python
def celsius_to_kelvin(celsius):
        return celsius + 273.15
``````

**Параметри:**
- 'fahrenheit' (float): температура

**Повертає:**
- 'float' - Фаренгейти переведені до цельсіїв

**Приклад:**
``````python
print(celsus_to_fahrenheit(10)) # Виведе: 50.00°F
