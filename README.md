# Temperature Converter
Simple temperature conversion tool.

## Функціональність
- Цельсії в фаренгейти
- Фаренгейти в цельсії
- Цельсії в кельвіни 
- Кельвіни в цельсії
- Фаренгейти в кельвіни

## Всановлення
1. Клонуйте репозиторій:
``````bash
git clone https://github.com/VladislavNeBobr/temperature-converter.git
``````
2. Перейдіть в папку проєкту
3.Запустіть програму:
``````bash
python temperature-converter.py
``````
